Slackbot Multiline Emoji Maker

A small tool to create mulitline Slackbot emojis!

This will be expanded to allow up to any amount of squares, processing directly from images and develop the image download - but for now enjoy!

How to use:

  1. You need to go to Image Splitter and cut the image into a 2x2 grid with a prefix that you can type out.
  2. You need to download the files and add them to your Slack workspace in the {workspace}.slack.com/customize/emoji section.
  3. You then need to enter the command you want to use and all four image names into the form inputs in their corresponding slots.
  4. Hit generate!
  5. Copy that string into your Slackbot responds section in a response in {workspace}.slack.com/customize/slackbot.
